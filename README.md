# NTU_EECP_DEMO
所需環境:Python 3.7  
安裝套件:pygame  
直接執行py檔，遊戲即會進行。  
但須注意 candy_crush .py 跟 Candy_crush素材 、music 需在同一資料夾下，路徑才會對。  

說明:遊戲規則跟效果都跟Candy Crush 一樣。視窗右下角有分數跟目標，還有可以走的步數。點擊第一次糖果時，右下角會出現白色方塊，這時候才可以點第二次。  
當分數高於目標或者步數用完時，遊戲即結束。
